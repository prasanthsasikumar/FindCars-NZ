Static JSON API — Data is pre-computed and served directly from static files. Updated daily via GitHub Actions. No authentication required.

Base URL

https://findcars.prasanthsasikumar.com/api/v1

Endpoints

GET /stats/overview

Overview statistics from the last 30 days.

{
  "total_listings": 20490,
  "unique_vehicles": 3149,
  "manufacturers": 94,
  "avg_price": 605.60,
  "median_price": 500.00,
  "top_manufacturers": { "Toyota": 4365, "Nissan": 2382 },
  "fuel_types": { "Petrol": 15215, "Diesel": 2277 },
  "registration_status": { "No": 16847, "Yes": 3643 }
}
GET /stats/price-trends

Historical price trends sampled every 7 days.

[
  { "date": "2026-07-08", "avg_price": 605.60, "median_price": 500.00, "count": 952 }
]
GET /manufacturers

Top 50 manufacturers by listing volume (last 30 days).

[
  { "manufacturer": "Toyota", "avg_price": 3200.50, "count": 4365, "unique_models": 45 }
]
GET /damage-analysis

Frequency counts of damage keywords.

{
  "Front Damage": 12500,
  "Rear Damage": 10200,
  "Airbags Deployed": 3800
}
GET /price-distribution

Vehicle count by price brackets.

{
  "$0-$500": 8466,
  "$500-$1k": 6933,
  "$1k-$2k": 2671,
  "$2k-$5k": 628,
  "$5k-$10k": 71,
  "$10k+": 15
}
GET /search

Latest day's full listings as JSON. Client-side filter by manufacturer, model, or price.

Quick Start

# Python
import requests
r = requests.get("https://findcars.prasanthsasikumar.com/api/v1/stats/overview")
print(f"Average price: ${r.json()['avg_price']}")
// JavaScript
fetch("https://findcars.prasanthsasikumar.com/api/v1/stats/overview")
  .then(r => r.json())
  .then(d => console.log("Average price: $" + d.avg_price));
