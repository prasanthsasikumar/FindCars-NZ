Static JSON API — All data is pre-computed as static JSON files. Served directly — no server-side processing. Updated daily via GitHub Actions. No authentication required.

Base URL

https://findcars.prasanthsasikumar.com/api/v1

Endpoints

GET /stats/overview

Aggregated stats from the last 30 days.

{
  "total_listings": 20490,  "unique_vehicles": 3149,  "manufacturers": 94,
  "avg_price": 605.60,      "median_price": 500.00,
  "top_manufacturers": { "Toyota": 4365, "Nissan": 2382, "Mazda": 2201 },
  "fuel_types": { "Petrol": 15215, "Diesel": 2277, "Hybrid": 2232 },
  "registration_status": { "No": 16847, "Yes": 3643 }
}
GET /stats/price-trends

Price history sampled every 7 days.

[ { "date": "2026-07-08", "avg_price": 605.60, "median_price": 500.00, "count": 952 } ]
GET /manufacturers

Top 50 manufacturers by volume (last 30 days).

[ { "manufacturer": "Toyota", "avg_price": 3200.50, "count": 4365, "unique_models": 45 } ]
GET /damage-analysis

Keyword frequency in damage descriptions.

{ "Front Damage": 12500, "Rear Damage": 10200, "Airbags Deployed": 3800 }
GET /price-distribution

Vehicle count by price bracket.

{ "$0-$500": 8466, "$500-$1k": 6933, "$1k-$2k": 2671, "$2k-$5k": 628, "$5k-$10k": 71, "$10k+": 15 }
GET /search

Full latest-day listings as JSON. Filter client-side by manufacturer, model, or price.

Quick Start

# Python
import requests
r = requests.get("https://findcars.prasanthsasikumar.com/api/v1/stats/overview")
print(r.json()["avg_price"])  # average price across last 30 days

# JavaScript
fetch("https://findcars.prasanthsasikumar.com/api/v1/manufacturers")
  .then(r => r.json())
  .then(d => d.slice(0, 5).forEach(m => console.log(m.manufacturer, m.count)));

About This Project

This site tracks damaged vehicle auction listings from Manheim New Zealand with automated daily scraping via GitHub Actions. Over 991 days and 460,000+ records, it provides price trends, manufacturer analytics, and a fully open-source REST API for programmatic access. Built with Python, Pandas, and Chart.js — hosted entirely on Netlify as a static site.
