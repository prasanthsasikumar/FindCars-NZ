REST API

Free, no auth required. All endpoints serve pre-computed static JSON — updated daily. Base: https://findcars.prasanthsasikumar.com/api/v1
GET/stats/overview

Aggregated stats from the last 30 days.

GET/stats/price-trends

Price history sampled every 7 days.

GET/manufacturers

Top 50 manufacturers by volume.

GET/damage-analysis

Damage keyword frequency.

GET/price-distribution

Vehicles by price bracket.

GET/search

Latest day's full listings as JSON.

Quick Start
# Python
import requests; r = requests.get("https://findcars.prasanthsasikumar.com/api/v1/stats/overview")
print(r.json()["avg_price"])
// JavaScript
fetch("https://findcars.prasanthsasikumar.com/api/v1/manufacturers")
  .then(r => r.json()).then(d => d.slice(0,5).forEach(m => console.log(m.manufacturer,m.count)))

About

This project tracks Manheim New Zealand damaged vehicle auctions with automated daily scraping via GitHub Actions. Over 991 days and 460,000+ records, it provides price trends, manufacturer analytics, and a free REST API. Built with Python, Pandas, and Chart.js — hosted on Netlify as a fully static site. Open source on GitHub.
